# imgui/

This directory must contain Dear ImGui source files.

Run `scripts/setup.ps1` to download them automatically, or place them here manually
from https://github.com/ocornut/imgui (latest release).

Required files:
  imgui.h
  imgui.cpp
  imgui_internal.h
  imgui_draw.cpp
  imgui_tables.cpp
  imgui_widgets.cpp
  imgui_impl_win32.h
  imgui_impl_win32.cpp
  imgui_impl_dx11.h
  imgui_impl_dx11.cpp
  imconfig.h
  imstb_rectpack.h
  imstb_textedit.h
  imstb_truetype.h
